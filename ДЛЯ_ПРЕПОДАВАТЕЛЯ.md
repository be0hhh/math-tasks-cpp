# Инструкция для демонстрации проекта

## Быстрый запуск

1. **Сборка проекта:**
   ```
   собрать.bat
   ```

2. **Запуск программ:**
   ```
   запустить_все.bat
   ```

## Что демонстрировать

### Для каждой задачи показать:

1. **Режим 0 - Полное решение задачи из задания:**
   - Полный алгоритм решения
   - Пошаговые вычисления
   - Промежуточные результаты
   - Проверки

2. **Режим 1-N - Интерактивный режим:**
   - Ввод своих данных
   - Вычисление результатов

## Особенности реализации

### Задача 1: Неравенство Коши
- Вычисление корней n-й степени
- Проверка неравенства для любых натуральных чисел
- Показ промежуточных шагов

### Задача 2: Конус максимального объема
- Оптимизация через производную
- Вычисление оптимального угла φ
- Параметры конуса (радиус, высота, объем)
- Проверка производной в оптимальной точке

### Задача 3: Полином и производные
- Схема Горнера для вычисления значений
- Вычисление производных любого порядка
- Разложение в ряд Тейлора
- Все вычисления из задания: p(2), p'(1), p''(3), p'''(-1)

### Задача 4: Неравенство с a, b, c
- Вычисление левой и правой части
- Анализ случаев равенства
- Промежуточные вычисления каждого слагаемого

### Задача 5: Комплексные полиномы
- Схема Горнера для комплексных чисел
- Деление на квадратичный множитель
- Синтетическое деление
- Проверка остатка

## Технические детали

- Все программы используют C++17
- Сборка через CMake
- Модульная структура (каждая задача - отдельная библиотека)
- Тесты для каждой задачи
- Красивый вывод чисел (целые без .0000000)

## Структура кода

Каждая задача содержит:
- `*.h` - интерфейс класса
- `*.cpp` - реализация
- `main.cpp` - интерактивная программа с подробными объяснениями
- `test_*.cpp` - тесты
