# Математические задачи на C++

Проект содержит решения пяти математических задач с подробными алгоритмами и интерактивными программами.

## Задачи

### 1. Задача A.2: Неравенство Коши
Доказать неравенство: `1/(m^(1/n)) + 1/(n^(1/m)) > 1` для всех натуральных m, n.

### 2. Задача о конусе максимального объема
Из круга радиуса R вырезан сектор углом φ радиан. Из оставшейся части делается прямой круговой конус. Найти φ, при котором конус имеет максимальный объем.

### 3. Задача с полиномом
Дано: `p(x) = x^5 - 2x^4 + 3x^3 - x^2 + x - 2`

Найти:
- p(2)
- p'(1)
- p''(3)
- p'''(-1)
- Разложение по (x-1)^k, k=0,...,5

### 4. Задача 13: Неравенство
Для a, b, c >= 0 доказать неравенство:
`a³b + b³c + c³a >= a²bc + b²ca + c²ab`

Найти случаи равенства.

### 5. Задача 5: Полином в комплексной точке
Вычислить алгебраический полином P_n(x) с вещественными коэффициентами в комплексной точке z₀ = u₀ + iv₀.

Разделить полином на `(x-z₀)(x-ẑ₀) = x² + px + q`, где `p = -2u₀`, `q = u₀² + v₀²`.


Каждая задача содержит:

- `*.cpp` - реализация методов
- `main.cpp` - интерактивная программа
- `test_*.cpp` - тесты

## Сборка и запуск

### Требования
- C++ компилятор с поддержкой C++17
- CMake 3.10 или выше
- Windows (для .bat скриптов) или Linux/macOS (нужно адаптировать скрипты)

### Сборка проекта

```bash
# Windows
собрать.bat

# Linux/macOS
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```

### Запуск программ

```bash
# Windows
запустить_все.bat

# Linux/macOS
cd build/Release
./run_task1
./run_task2
# и т.д.
```

## Особенности

- **Подробные алгоритмы**: Каждая программа содержит полное объяснение алгоритма решения с пояснениями "зачем и почему"
- **Промежуточные вычисления**: Показываются все шаги вычислений
- **Схема Горнера**: Используется для эффективного вычисления полиномов
- **Красивый вывод**: Целые числа выводятся без лишних нулей
- **Интерактивный режим**: Можно вводить свои данные для проверки

## Режимы работы

Каждая программа имеет два режима:
- **0 - РЕШЕНИЕ ЗАДАЧИ ИЗ ЗАДАНИЯ**: Полный алгоритм + подробные вычисления
- **1-N - Интерактивный режим**: Ввод своих данных и вычисление результатов
