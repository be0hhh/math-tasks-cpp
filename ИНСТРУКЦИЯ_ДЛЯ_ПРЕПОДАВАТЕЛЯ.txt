1. ЗАПУСК ПРОЕКТА:
   - Дважды кликните: собрать.bat
   - Дождитесь завершения сборки
   - Дважды кликните: запустить_все.bat

2. ДЕМОНСТРАЦИЯ КАЖДОЙ ЗАДАЧИ:
   
   Для каждой задачи (1-5) показать:
   
   а) Режим 0 - "РЕШЕНИЕ ЗАДАЧИ ИЗ ЗАДАНИЯ":
      - Полный алгоритм решения
      - Пошаговые вычисления
      - Промежуточные результаты
      - Проверки
   
   б) Режим 1 - Интерактивный режим:
      - Ввод своих данных
      - Вычисление результатов

3. ОСОБЕННОСТИ ПРОЕКТА:
   - Все 5 задач полностью реализованы
   - Подробные алгоритмы с объяснениями
   - Промежуточные вычисления на каждом шаге
   - Схема Горнера в задачах 3 и 5
   - Красивый вывод (целые числа без .0000000)
   - Модульная структура кода
   - Тесты для каждой задачи

   Все файлы загружены и доступны публично.

5. СТРУКТУРА:
   - Каждая задача в отдельной папке
   - Заголовочные файлы (.h)
   - Реализация (.cpp)
   - Интерактивные программы (main.cpp)
